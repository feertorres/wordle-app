# Wordle App

Clon del juego Wordle en español, con tema oscuro y paleta de colores personalizada.

🔗 **[Jugar ahora](https://feertorres.github.io/wordle-app/)**

## Cómo jugar

- Adiviná la palabra oculta de 5 letras en un máximo de **5 intentos**
- Después de cada intento, el color de las letras indica qué tan cerca estás:
  - 🟥 **Rosa** — la letra está en la posición correcta
  - 🟧 **Naranja** — la letra está en la palabra pero en otra posición
  - ⬛ **Gris** — la letra no está en la palabra
- Hay una palabra nueva cada **5 minutos**

## Tecnologías

- [React 17](https://reactjs.org/)
- [Tailwind CSS 3](https://tailwindcss.com/)
- [Headless UI](https://headlessui.com/) — modales
- [Heroicons](https://heroicons.com/) — íconos

## Desarrollo local

```bash
npm install
npm start       # inicia en http://localhost:3000
npm run build   # genera el build de producción
```
